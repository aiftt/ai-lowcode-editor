## 技术栈

- Next.js 15
- React 19
- TypeScript
- Zustand (状态管理)
- TailwindCSS 4
- 自定义拖拽实现

## 核心特点

- 无iframe的真正所见即所得编辑体验
- 自定义高性能拖拽实现
- 统一渲染引擎支持编辑-预览-发布无缝切换
- AI辅助设计与开发
- 高度可扩展的组件系统
